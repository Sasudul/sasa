##  Overview

Welcome to the repository of my personal portfolio! This project is crafted to provide a butter-smooth user experience, utilizing **Lenis** for smooth scrolling, **GSAP** and **Framer Motion** for complex, highly performant animations, and **Tailwind CSS** for a pixel-perfect, responsive design.

##  Key Features

- ** Blazing Fast Performance:** Powered by Vite & React 19.
- ** Stunning Animations:** Advanced scroll-triggered and interactive animations using GSAP & Framer Motion.
- ** Fully Responsive:** Beautifully adapts to mobile, tablet, and desktop screens using Tailwind CSS.
- ** Type-Safe:** Fully developed in **TypeScript** (91.3% of the codebase) for reliable, robust code.
- ** Smooth Scrolling:** Integrated with Lenis for a premium, frictionless scrolling experience.
- **Modern UI:** Clean, intuitive, and modern interface featuring `lucide-react` icons.

## ⚙️ Local Setup & Installation

Want to run this project locally? Follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sasudul/Sasundul-Portfolio.git
   ```

2. **Navigate into the project directory**
   ```bash
   cd Sasundul-Portfolio/sasundul-portfolio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
   *The app will be running at `http://localhost:5173`.*
